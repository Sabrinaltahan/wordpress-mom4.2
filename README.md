Realestate Theme - WordPress Projekt


## 🚀 Projektbeskrivning
Ett anpassat WordPress-tema för ett fastighetsföretag, med:
- Dynamisk startsida med **Hero-sektion** och nyhetssektion
- Om-sida med anpassad mall
- **Responsiv** navigationsmeny för dator och mobil
- Användning av **HTML, CSS, JavaScript** för gränssnittsdesign

---

## ⚙️ Funktioner
- Visa nyheter (Posts) med utvalda bilder
- Hamburger-meny för mobilvisning
- Lätt att lägga till nya sidor
- Sidlayout som fungerar med företagets tema

---

## 📂 Projektstruktur
company-theme/
├─ style.css        # Stilfiler
├─ functions.php    # Registrera scripts och styles
├─ index.php        # Startsida
├─ page.php         # Mall för statiska sidor
├─ header.php       # Gemensam header
├─ footer.php       # Gemensam footer
├─ main.js          # JavaScript-funktioner
└─ images/          # Bilder


## Installation
1. Placera mappen `realestate-theme` i `wp-content/themes/`
2. Aktivera temat via WordPress-adminpanelen
3. Skapa sidor (Hem,nyheter,kontakt och Ai-verktyg)
4. Lägg till några inlägg för att visas i nyhetssektionen
5. Testa navigationsmenyn på dator och mobil
